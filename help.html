TabStore - Help & Guide

Features

  • Save All Tabs: Save all open tabs in your current window as a named group (accordion) with the date.
  • Restore Tabs: Instantly open all URLs from a saved group in new tabs.
  • Delete Groups: Remove any saved group of URLs with confirmation dialog.
  • Download URLs: Download the URLs of a group as an Excel (.xlsx) file. The export is done with true Excel XML formatting, ensuring best compatibility with Microsoft Excel and Google Sheets.
  • Multi URLs Opener: Paste multiple URLs and open them all at once.
    • Preserve URLs: Option to keep the pasted URLs for next time using local storage. When unchecked, previously saved URLs are cleared from storage automatically.
    • Extract URLs: Use the Extract URLs from text button to parse any text and pull out valid URLs automatically (smart regex-based parsing).
    • Open in Group: Open all URLs in a new Chrome tab group with random color assignment. (Cannot be used with Incognito or New Window mode; selection disables the other two options.)
    • Open in Incognito: Open in Incognito mode. Mutually exclusive—disables Group and New Window options if checked.
    • Open in New Window: Open in a new regular browser window, mutually exclusive with the other two options.
    • Delete All: Clear all pasted URLs from textarea and local storage.
    • Duplicate URLs Removal: Duplicate URLs are automatically ignored and only opened once.
    • Mutual Exclusion/Disabling: The three "Open in..." checkboxes are mutually exclusive: selecting one disables the others for clarity and to prevent conflicts.
  • URLs Scheduler: Schedule URLs to open automatically at specific times.
    • One Time Scheduler: Open URLs once—date and time pickers default to the current date and time, and are reset to 'now' if canceled before confirm.
    • Daily Scheduler: Schedule recurring launches at a chosen time each day—defaults to the current time every time the picker is shown or canceled.
    • URLs Preservation: Scheduled URLs are auto-saved and restored as you type or open the scheduler.
    • Date/Time Validation: Prevents scheduling in the past and validates required time fields.
    • Cancel Options: Cancel at any time before confirming, reverting the pickers to their defaults.
  • Random Tab Group Color: When opening URLs in a group, a random color is chosen from 9 available Chrome group colors for clarity.
  • Pin/Unpin All Tabs: Instantly (un)pin all tabs in your window.
  • Reload Tabs: Reload all tabs in your window with normal or cache-bypassing hard reload.
  • Close Tabs (except current): Close all tabs except your current, with tooltip guidance in the UI.
  • Keyboard Shortcuts: Use keyboard combinations (Ctrl+Shift+Y and Ctrl+Shift+S by default; see Shortcuts section) to quickly open popup or autosave all tabs.
  • Help Button: The popup's help button opens this guide in a new tab at any time.
  • Local Storage: All data and user preferences are kept entirely in your browser and never shared externally.

How to Use

  1. Saving Tabs
    • Go to the Save Tab's URLs tab in the extension popup.
    • Click Add URLs to save all open tabs. You will be prompted to name the group.
    • Your saved groups will appear as accordions. Expand any group for options to open all URLs, download to Excel, or delete the group (with confirmation).
  2. Restoring Tabs
    • Click the button on a group to open all URLs in new tabs.
  3. Downloading URLs
    • Click the button to download the group's URLs as an Excel file with standard Excel XML formatting. The file is compatible with Microsoft Excel, Google Sheets, and others.
  4. Deleting a Group
    • Click the button to remove a group. Confirmation ensures accidental deletions are avoided.
  5. Multi URLs Opener
    • Go to the Multi URLs Opener tab.
    • Paste or type URLs (one per line) in the textarea. Use Extract URLs to automatically parse and pull out URLs from any copied/pasted text.
    • Click Open All to open all listed URLs (duplicates are ignored).
    • Use checkboxes for Preserve URLs (auto-restores/culls URLs from storage), Open in new group (random color, can’t combine with others), Incognito, or new window. Only one of these three "Open in..." options may be selected at a time.
    • If you check one of those three, the other two are disabled for mutual exclusion clarity.
    • Click Delete All to clear textarea and storage instantly.
  6. URLs Scheduler
    • Go to the URLs Scheduler tab.
    • Paste/type URLs (one per line) into the textarea (auto-preserved to storage).
    • One Time Schedule: Click Schedule One Time to show date/time picker (defaults to NOW and resets to current if canceled). Choose a future date and time, then Confirm Schedule to save.
    • Daily Schedule: Click Schedule Daily to show time picker (defaults to current time and resets if canceled). Pick your recurring time, then Confirm Schedule to set it.
    • Cancel at any point before confirming to revert to safe defaults for scheduling.
    • All times and schedules are validated: you cannot select a past date/time.
  7. Reload Tabs
    • Switch to Additional Options. Choose Normal Reload to update all tabs (normal cache), or Hard Reload to force tabs to reload freshly (bypasses browser cache).
  8. Close Tabs (except current)
    • Click Close all tabs to close every tab in the window except for your currently active tab. A tooltip in the UI explains this clearly for reference.
  9. Help Guide
    • Click the guide icon in the popup’s header at any time to open this help/guide in a new tab.
  10. Other Options
    • Pin/Unpin all tabs in your window instantly using the Pin All or Unpin All buttons under Additional Options.
  11. Keyboard Shortcuts
    • Press Ctrl+Shift+Y to open the popup quickly. Press Ctrl+Shift+S to instantly save all current tabs without prompting (name is auto-generated as "Quick Save_X").

Shortcuts

  • Open Popup: Ctrl+Shift+Y - Opens the TabStore extension popup
  • Save All Tabs: Ctrl+Shift+S - Quickly saves all current tabs with auto-generated name "Quick Save_X"

Notes

  • All data is stored locally in your browser using Chrome's local storage API.
  • Extension does not sync or upload your URLs anywhere—complete privacy protection.
  • When opening multiple URLs, duplicate entries are automatically ignored and only opened once.
  • "Open URLs in new group", "Open URLs in Incognito", and "Open URLs in new window" are mutually exclusive—selecting one will disable the others automatically.
  • Scheduled URLs are stored using Chrome's alarms API and will execute even if the browser is closed (when reopened).
  • One-time scheduled alarms are automatically cleaned up after execution.
  • Daily scheduled alarms repeat every 24 hours from the scheduled time.
  • URLs in scheduled textarea are automatically saved as you type and restored when reopening the extension.
  • Excel downloads use proper XML formatting for compatibility with Microsoft Excel and other spreadsheet applications.
  • Hard reload bypasses browser cache and forces fresh content from servers—useful for development and troubleshooting.
  • Extension automatically opens this help page when first installed to guide new users.
  • All checkbox states and preferences are preserved between extension sessions.
