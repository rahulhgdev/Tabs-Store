TabStore - Help & Guide

Features

  • Save All Tabs: Save all open tabs in your current window as a named group (accordion) with the date.
  • Restore Tabs: Instantly open all URLs from a saved group in new tabs.
  • Delete Groups: Remove any saved group of URLs with confirmation dialog.
  • Download URLs: Download the URLs of a group as an Excel (.xlsx) file with proper formatting.
  • Multi URLs Opener: Paste multiple URLs and open them all at once.
    • Preserve URLs: Option to keep the pasted URLs for next time using local storage.
    • Open in Group: Option to open all URLs in a new Chrome tab group with random color assignment. (Cannot be used with Incognito mode.)
    • Open in Incognito: Option to open all URLs in Incognito mode. (Cannot be used with Group mode.)
    • Open in New Window: Option to open all URLs in a new browser window. (Cannot be used with Group or Incognito mode.)
    • Delete All: Clear all pasted URLs from textarea and storage.
    • Duplicate URLs Removal: Duplicate URLs are automatically ignored and only opened once when using this feature.
    • Mutual Exclusion: "Open in Group", "Open in Incognito", and "Open in New Window" are mutually exclusive—selecting one disables the others.
  • URLs Scheduler: Schedule URLs to open automatically at specific times.
    • One Time Scheduler: Schedule URLs to open once at a specific date and time in the future.
    • Daily Scheduler: Schedule URLs to open daily at a specific time.
    • URLs Preservation: Scheduled URLs are automatically saved and restored when reopening the extension.
    • Date/Time Validation: Prevents scheduling in the past and validates input fields.
    • Cancel Options: Cancel scheduled operations before confirmation.
  • Random Tab Group Color: When opening URLs in a group, a random color is chosen from 9 available colors for the Chrome tab group.
  • Pin All Tabs: Instantly pin all tabs in your current window.
  • Unpin All Tabs: Instantly unpin all tabs in your current window.
  • Reload Tabs: Reload all tabs in your current window. You can choose normal reload or hard reload (bypass cache).
  • Close Tabs (except current): Close all tabs except your currently active tab. Tooltip in the extension provides more info.
  • Keyboard Shortcuts: Quick access to extension features using keyboard combinations.
  • Help Button: The help button in the popup opens this guide page in a new tab.
  • Local Storage: All data is stored locally in your browser with no external data transmission.

How to Use

  1. Saving Tabs
    • Go to the Save Tab's URLs tab in the extension popup.
    • Click Add URLs to save all open tabs. You will be prompted to name the group.
    • Your saved groups will appear as accordions. Expand to see URLs.
  2. Restoring Tabs
    • Click the button on a group to open all URLs in new tabs.
  3. Downloading URLs
    • Click the button to download the group's URLs as an Excel file.
  4. Deleting a Group
    • Click the button to remove a group.
    • You will be asked to confirm before a group is deleted, to prevent accidental removal.
  5. Multi URLs Opener
    • Go to the Multi URLs Opener tab.
    • Paste or type URLs (one per line) in the textarea.
    • Click Open All URLs to open them. Duplicates are ignored by default.
    • Check Preserve URLs to keep your list for next time using local storage.
    • Check Open URLs in new group to open all in a Chrome tab group with random color. This cannot be used together with Incognito or New Window mode.
    • Check Open URLs in Incognito to open all in Incognito mode. This cannot be used together with Group or New Window mode.
    • Check Open URLs in new window to open all in a new browser window. This cannot be used together with Group or Incognito mode.
    • Click Delete All URLs to clear the textarea and storage.
  6. URLs Scheduler
    • Go to the URLs Scheduler tab.
    • Paste or type URLs (one per line) in the textarea. URLs are automatically saved to storage.
    • One Time Scheduler:
      • Click Schedule One Time to show date/time picker.
      • Select a future date and time.
      • Click Confirm Schedule to set the alarm.
      • Use Cancel to abort the scheduling process.
    • Daily Scheduler:
      • Click Schedule Daily to show time picker.
      • Select a time for daily execution.
      • Click Confirm Schedule to set the recurring alarm.
      • Use Cancel to abort the scheduling process.
  7. Reload Tabs
    • Go to the Additional Options tab.
    • Click Normal Reload to reload all tabs normally.
    • Click Hard Reload to reload all tabs from the server and bypass the cache.
  8. Close Tabs (except current)
    • Click Close all tabs to close every tab except the one you're currently using. Hover the info icon to see this tip as a tooltip.
  9. Help Guide
    • Click the help icon at the top right of the popup to open this Help & Guide page at any time.
  10. Other Options
    • Click Pin All Tabs to pin every tab in your current window.
    • Click Unpin All Tabs to unpin every tab in your current window.

Shortcuts

  • Open Popup: Ctrl+Shift+Y - Opens the TabStore extension popup
  • Save All Tabs: Ctrl+Shift+S - Quickly saves all current tabs with auto-generated name "Quick Save_X"

Notes

  • All data is stored locally in your browser using Chrome's local storage API.
  • Extension does not sync or upload your URLs anywhere—complete privacy protection.
  • When opening multiple URLs, duplicate entries are automatically ignored and only opened once.
  • "Open URLs in new group", "Open URLs in Incognito", and "Open URLs in new window" are mutually exclusive—selecting one will disable the others automatically.
  • When opening URLs as a group, a random color is assigned from 9 available colors (grey, blue, red, yellow, green, pink, purple, cyan, orange) to the Chrome tab group for visual organization.
  • Scheduled URLs are stored using Chrome's alarms API and will execute even if the browser is closed (when reopened).
  • One-time scheduled alarms are automatically cleaned up after execution.
  • Daily scheduled alarms repeat every 24 hours from the scheduled time.
  • URLs in scheduled textarea are automatically saved as you type and restored when reopening the extension.
  • Excel downloads use proper XML formatting for compatibility with Microsoft Excel and other spreadsheet applications.
  • Hard reload bypasses browser cache and forces fresh content from servers—useful for development and troubleshooting.
  • Extension automatically opens this help page when first installed to guide new users.
  • All checkbox states and preferences are preserved between extension sessions.
